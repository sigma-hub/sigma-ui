## Installation

The Date Picker is a composition of: `<Popover />` and either `<Calendar />` or `<RangeCalendar />`

Install them to create Date Picker:

- [Popover](./popover)
- [Calendar](./calendar)
- [Range Calendar](./range-calendar)
